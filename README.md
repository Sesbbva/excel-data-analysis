## Excel Data Analysis – Blue Lake Sports

### Overview
This project analyzes transaction data for a sporting goods company using Microsoft Excel.  
The goal was to transform raw data into meaningful insights using data cleaning, analysis, and visualization techniques.


### Key Tasks

- Imported raw data from a text file into Excel  
- Cleaned and structured the dataset into a usable table  
- Filtered and sorted data to focus on key categories  
- Built a PivotTable to analyze transactions by department  
- Created a PivotChart to visualize results  
- Designed a Sunburst chart to show hierarchical relationships (City → Department)  
- Applied Excel formulas such as XLOOKUP and SUMPRODUCT


### Key Insights

- Game Room and Running departments show high transaction volume  
- Transaction distribution varies significantly across cities  
- Certain product categories dominate within specific regions
